## Overview

An AI-powered product-identification platform that transforms visual product recognition
using OCR and machine-learning techniques. Users can identify products from images, search a
structured product database, and explore demographic insights — all behind a clean,
responsive interface.

## Key Features

- **AI product recognition:** Identify products from images via OCR + ML.
- **Guest access:** Try the core experience without creating an account.
- **Accounts & history:** User authentication and account management.
- **Searchable database:** Product catalogue with search capabilities.
- **Data visualization:** Demographic insights and analytics.
- **Responsive UI:** Dark/light mode and mobile-friendly layout.

## Tech Stack

- **TypeScript** full-stack application
- **OpenAI** API for AI-powered identification
- **PostgreSQL** database with migrations
- **Docker** for reproducible deployment
