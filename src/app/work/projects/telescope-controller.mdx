## Overview

Telescope Controller is a web-based dashboard for operating telescopes and organizing
observing runs from the browser. It provides a clean operator interface over the lower-level
instrument control used at the Center for Astronomy, Space Science and Astrophysics (CASSA),
letting observers manage telescope locations, build observation plans, and monitor sessions
without touching the command line.

## Key Features

- **Multi-location dashboard:** Register and switch between telescope locations from a single
  control surface.
- **Observation plans:** Create, schedule, and track observation plans and their progress.
- **Live session view:** A responsive dashboard for monitoring the active observing run.
- **Modern web stack:** Fast, responsive UI built for use in the field and remotely.

## Technologies Used

- **Next.js** + **TypeScript**
- Component-driven, responsive UI deployed on Vercel
