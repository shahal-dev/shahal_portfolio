## Overview

Assignment Harbor is a platform for students to get help with their assignments from verified
experts. It pairs a student-facing request flow with expert matching, handling authentication,
notifications, and the full request lifecycle.

## Key Features

- **Expert matching:** Connect students with verified experts for assignment help.
- **Secure auth:** JWT-based authentication and account management.
- **Email notifications:** Transactional email via SMTP for updates and verification.
- **Full request lifecycle:** From posting a request to delivery.

## Tech Stack

- **TypeScript** full-stack application
- **PostgreSQL** database
- **JWT** authentication, **SMTP** email
- **Docker / Docker Compose** for deployment
