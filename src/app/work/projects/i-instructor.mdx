## Overview

i-Instructor is a peer-to-peer learning platform that connects students with qualified
instructors for instant academic support — any time of day. Learners get matched with an
instructor in minutes and collaborate over real-time chat, audio/video, and a shared
whiteboard.

## Key Features

- **Instant help:** Get connected with an instructor in roughly two minutes.
- **Many subjects:** Mathematics, Computer Science, Physics, Chemistry, Biology, English,
  and more.
- **Real-time collaboration:** Text, audio, and video chat with a shared whiteboard.
- **Flexible pricing:** Pay-as-you-go or monthly subscription plans.
- **Knowledge base:** Community-driven solutions and guides.
- **Two roles:** Learners get help and track sessions; instructors teach, manage sessions,
  and track earnings.

## Tech Stack

- **Frontend:** React 18 + TypeScript + Vite, Tailwind CSS
- **Auth:** Firebase Authentication with Google OAuth
- **Database:** Firebase Firestore
- **State:** React Context API
