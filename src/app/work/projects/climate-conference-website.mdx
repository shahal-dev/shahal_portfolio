## Overview

A modern, responsive website for the conference *"Environment and Changing Climate: Global
Strategies and Local Actions"* at Independent University, Bangladesh. The site presents the
programme, events, and schedule with an environmental-themed, glass-morphism design.

## Key Features

- **Environmental-themed design:** A polished, on-brand visual identity for the conference.
- **Interactive events carousel:** Browse sessions and highlights.
- **Conference timeline:** A dedicated page for the schedule.
- **Fully responsive:** Works cleanly across devices.

## Tech Stack

- **React 18** + **TypeScript**
- **Tailwind CSS**, **Lucide** icons
- **Vite** build tooling
- **Docker + Nginx** for deployment
