## Overview

AgroLink is a web platform built to connect farmers with the resources, information, and
market access they need. The goal is to bring digital tooling to the agricultural value chain
— making it easier for farmers to plan, sell, and access support.

## Key Features

- **Farmer-centric platform:** Designed around the needs of agricultural users.
- **Resource access:** Surfaces relevant agricultural information and services.
- **Market linkage:** Helps connect producers with buyers and resources.

## Tech Stack

- Modern full-stack web application
- Responsive, accessible UI
